# 🕸️ Академия LangGraph: Интерактивный учебник

Добро пожаловать в проект обучающего веб-приложения на базе **Streamlit**. Этот проект разработан для того, чтобы наглядно и интерактивно обучать разработчиков тонкостям создания AI-агентов с помощью фреймворка [LangGraph](https://python.langchain.com/docs/langgraph/).

В приложении собраны лучшие практики работы с графовыми архитектурами, а также примеры интеграции с фреймворками **LangChain** и **PydanticAI**.

## 🚀 Как запустить

Вам понадобится Python версии 3.9 или выше. Выполните следующие шаги в терминале:

1. Перейдите в директорию проекта:
   ```bash
   cd d:\tatia\langgraph_tutorial_app
   ```

2. Установите все зависимости (Streamlit, LangGraph, LangChain и др.):
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите учебное приложение:
   ```bash
   streamlit run app.py
   ```

4. Ваш браузер автоматически откроет локальную страницу по адресу `http://localhost:8501`.

## 📚 Программа курса (Включенные уроки)

Интерфейс учебника разделен на 4 практических урока (доступны через боковое меню приложения):

* **Урок 1: Введение в StateGraph**
  Понимание базовой механики. Как объявлять состояние (State) в виде `TypedDict`, как создавать функции-узлы (Nodes) и выстраивать их в единый конвейер.
* **Урок 2: Редьюсеры (Управление состоянием)**
  Ключевой концепт агентов — долговременная память. Изучение того, как `operator.add` позволяет сохранять историю сообщений, а не перезаписывать её.
* **Урок 3: Маршрутизация (Conditional Edges)**
  Превращаем линейный скрипт в "умного" агента, который умеет принимать решения и переходить в специализированные ветви графа в зависимости от контекста (например, "ветвь успеха" или "ветвь исправления ошибки").
* **Урок 4: Интеграция Навыков (Tools)**
  Подключение внешних навыков (поиск, базы данных, API) к агенту. Демонстрация использования пребилда `ToolNode` из экосистемы **LangChain** и создание Wrapper-узлов для строго типизированных агентов на **PydanticAI**.
