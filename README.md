# 🕸️ Академия LangGraph: Интерактивный учебник

Добро пожаловать в проект обучающего веб-приложения на базе **Streamlit**. Этот проект разработан для того, чтобы наглядно и интерактивно обучать разработчиков тонкостям создания AI-агентов с помощью фреймворка [LangGraph](https://python.langchain.com/docs/langgraph/).

В приложении собраны лучшие практики работы с графовыми архитектурами, а также примеры интеграции с фреймворками **LangChain** и **PydanticAI**.

## 🚀 Как запустить

Вам понадобится Python версии 3.9 или выше. Выполните следующие шаги в терминале:

1. Перейдите в директорию проекта:
   ```bash
   cd langgraph_tutorial_app
   ```

2. Установите все зависимости (Streamlit, LangGraph, LangChain и др.):
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите учебное приложение:
   ```bash
   streamlit run app.py
   ```

4. Ваш браузер автоматически откроет локальную страницу по адресу `http://localhost:8501`.

## ⚙️ Настройки и Конфигурация (.env)
Обучающие уроки (с 1 по 18) работают 100% оффлайн. Однако, **Урок 19 (Интерактивный Экзамен)** использует живые нейросети (LLM) для автоматической генерации тестовых заданий и оценки понимания материала.

Для этого потребуется бесплатный ключ от **OpenRouter**.
1. Скопируйте файл `.env.example` в новый файл `.env` в корне проекта.
2. Откройте `.env` и вставьте ваш ключ от OpenRouter.

### 🤖 Выбор моделей (OpenRouter)
В файле `.env` вы можете задать модель по умолчанию для Экзамена (параметр `OPENROUTER_DEFAULT_MODEL`), например:
* `z-ai/glm-4.5-air:free`
* `liquid/lfm-40b:free`
* `google/gemma-2-9b-it:free`

В приложении (Урок 19) реализован **умный кэширующий загрузчик**, который делает запрос к API OpenRouter при каждом запуске и автоматически подтягивает в выпадающий список все самые свежие бесплатные модели (`free` tier). Вам не нужно жестко прописывать их в коде!
